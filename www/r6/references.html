CSS and XHTML References

Library

Flying Saucer makes use of the CSSParser project on Sourceforge, formerly the Steady State CSS parser. All of our modifications to it have been contributed back to this project per the LGPL license.
The Specs:

  • W3C XHTML specification
  • CSS 2.1 Specification (exhaustive but precise)
  • W3CSchools XHTML Website a more hands on definition of XHTML with examples and tutorials

Learning about Good CSS Design

  • A List Apart years of articles about good CSS design
  • CSS Edge demos and cool tutorials of advanced CSS
  • CSS Zen Garden An example site showing the power of CSS.
  • Dashboard Apple's technology to free XHTML/CSS from the webbrowser.

Books

  • Eric Myers on CSS
  • CSS Pocket Reference, 2nd Edition (only 10$!)
