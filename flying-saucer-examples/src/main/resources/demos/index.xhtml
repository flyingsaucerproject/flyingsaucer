Flying Saucer

XML + CSS Renderer

The Flying Saucer Team is proud to present our 100% Java library for rendering XHTML + CSS2.1 including tables, forms, the box model, lists, selectors, images, and text formatting.

The Flying Saucer XHTMLRenderer can be used anywhere a program needs rich styled content such as ebook readers, chat apps, splash screens, help viewers, report generation, web-enabled thick clients, and online music stores.

This release brings updates to the infrastructure with:

  • rewritten layout and render engine
  • render direct to PDF, including embedded fonts, header, footer, page counters and bookmark support
  • pagination
  • z-indexing/layers
  • margin collapse
  • better float processing
  • more speed improvments
