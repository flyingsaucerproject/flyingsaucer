Flying Saucer

The 100% Java XHTML/CSS2 Renderer

 
 

What is it?

Flying Saucer is a Java library which lets you embed XHTML inside of your application. Anywhere you use styled text you can use Flying Saucer.

What can I do with it?

You can use Flying Saucer inside all sorts of programs like:

  • Online ebook and help reader
  • A webblog editor
  • Searchable dictionaries
  • An RSS reader
  • An attractive address book
  • A Help -> About menu
  • Report generation
  • An online music store
  • Your own imagination

What does it support?

We cover most features from the CSS 2.1 spec. We have put together a number of demo pages for you to try out. Try some samples from the Demos menu above or by clicking on this link.

Where can I find out more?

Check us out online at http://xhtmlrenderer.dev.java.net/ and join the mailing list!
